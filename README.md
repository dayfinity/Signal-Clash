# Signal-Clash
learning interactive programming fundamentals. It is designed with simplicity in mind and uses only the standard library.

The game places the player in a competitive digital environment where different communication channels like instagram and telegram appear as symbolic signal sources during gameplay rounds. Each round challenges the player to choose the correct response path based on random system output.

The goal is to demonstrate how input handling, random events, and scoring logic work together in a structured program. The code is intentionally modular so beginners can easily modify behavior or extend functionality.
